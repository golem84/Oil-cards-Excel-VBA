Функционал позволяет формировать карточки для проб масла для общего анализа и анализа на класс чистоты в соответствии с руководством по эксплуатации на ГТЭС/ЭГЭС типа "Урал", а также ГТУ на базе авиационных двигателей разработки АО "ОДК-Авиадвигатель".

Функционал реализован в виде встроенного в документ Excel макроса.
Для работы макроса необходимо при открытии документа Excel разрешить работу макросов в предупреждающем сообщении. 
Работа макроса запускается по нажатию кнопки, расположенной на главном листе документа. По окончании работы макроса формируется сообщение пользователю о том, что работа закончена. В рабочей папке с текущим документом Excel появляются два документа Word, оформленные в соответствии с прилагаемыми шаблонами.
Для формирования карточки анализа из бочек в столбце "номер ГТЭС" указать "Бочка 1", наработку масла "0", остальные ячейки оставить пустыми.

Ячейки листа защищены от изменения кроме тех, в которых записываются данные для формирования карточек.

ВНИМАНИЕ:
1. Наименование ГТЭС используется для сохранения файлов в файловой системе, поэтому использование кавычек запрещено. Макрос выполняет проверку и, при нахождении в тексте кавычек, прекращает работу.
2. Для работы макроса используются шаблоны. Шаблоны должны находиться в одной папке с данным документом Excel. Шаблоны не удалять, не переименовывать, не открывать и не изменять - может нарушиться логика работы макроса.

Макрос защищен от просмотра и редактирования во избежание нарушения логики работы.
